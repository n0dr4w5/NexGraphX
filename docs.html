gui (title, width, height)

Sets up the GUI window with a title and dimensions.

gprint (text)

Prints the provided text in the GUI window.

gres (True or False)

Sets the GUI window's resizable property.

gui_background (color)

Sets the background color of the GUI window.

gui_text_color (color)

Sets the text color in the GUI window.

gui_text_font (font, size)

Sets the font for the text in the GUI window.

gui_dis ()

Disables the text box in the GUI window.

gprint_i (image)

Adds an image to the GUI window.

run ()

Starts the GUI event loop.

Usage Example

from pynexs import *

# Initialize GUI
gui("My App", 800, 600)

# Print text
gprint("Hello, world!")

# Set resizable
gres(True)

# Set background color
gui_background("#F0F0F0")

# Set text color
gui_text_color("red")

# Set text font
gui_text_font("Arial", 12)

# Disable text box
gui_dis()

# Add an image
gprint_i(image_object)

# Run the application
run()
